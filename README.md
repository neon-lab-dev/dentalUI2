## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables in `.env`
4. Start the development server:
   ```bash
   npm run dev
   ```

## Environment Variables
```
NEXT_PUBLIC_API_URL=http://localhost:8080
```
